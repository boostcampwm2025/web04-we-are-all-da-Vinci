# 앱인토스 수동 QA 체크리스트

> 자동화할 수 없는 항목입니다. 실기기(토스 샌드박스 앱)에서 점검하세요.

---

## 1. 실기기 테스트 (토스 샌드박스)

- [ ] 앱 진입 5초 이내 로드
- [ ] TDS 컴포넌트 정상 렌더링 (ProgressBar, Button, ConfirmDialog)
- [ ] TDSMobileAITProvider 정상 동작
- [ ] Safe area 적용 확인 (노치/하단바)
- [ ] 배너 광고 로드 및 표시 (memorize 화면)

## 2. 내비게이션 & 뒤로가기

- [ ] Android 하드웨어 뒤로가기 각 화면에서 정상 동작
- [ ] 홈에서 뒤로가기 시 미니앱 종료 (차단 바텀시트 없음)
- [ ] 드로잉 화면에서 뒤로가기 시 적절한 동작 (확인 다이얼로그 등)
- [ ] 무한 내비게이션 루프 없음
- [ ] 딥링크로 `/` 진입 정상 동작

## 3. 네트워크 장애

- [ ] 오프라인 시 흰 화면 크래시 없음
- [ ] API 타임아웃 시 에러 메시지 표시 (해요체)
- [ ] 광고 SDK 실패 시 메인 플로우 차단 없음
- [ ] 네트워크 복구 후 정상 동작

## 4. 백그라운드/포그라운드 전환

- [ ] 백그라운드 후 복귀 시 정상 동작
- [ ] 캔버스 드로잉 상태 유지
- [ ] 백그라운드에서 오디오 재생 없음
- [ ] 타이머 동작 시 백그라운드 처리 정상

## 5. 성능

- [ ] 캔버스 드로잉 시 버벅임 없음
- [ ] 메모리 안정적 (장시간 드로잉 세션)
- [ ] CPU 스파이크 없음
- [ ] 스크롤 성능 부드러움 (memorize/status 화면)

> 성능 측정/디버깅 도구 (일반 브라우저 DevTools 아님 — 샌드박스에 연결된 실제 WebView에서):
>
> - iOS(iPhone 샌드박스): Mac Safari → 개발자 메뉴 → iPhone WebView → Web Inspector
> - Android: Android Studio 에뮬레이터 + 앱인토스 Android 샌드박스 APK → Chrome `chrome://inspect`

## 6. 광고 수동 체크

- [ ] 배너 광고 지연 없이 프리로드
- [ ] 핵심 드로잉 플로우에 광고 없음
- [ ] 광고 닫기 시 크래시/시각 오류 없음
- [ ] 광고 노출 이벤트 로그 수집 확인 (TossAds 대시보드)

## 7. 시각/UX

- [ ] 모든 텍스트 배경 대비 가독성 확보
- [ ] 색상 팔레트 버튼 구분 가능
- [ ] 캔버스 터치 반응 지연 없음
- [ ] 툴바 아이콘 충분한 대비
- [ ] 가로 오버플로 없음 (모든 화면)
- [ ] 모든 문구 해요체 사용

## 8. 화면 크기

- [ ] 소형 화면 (iPhone SE) 레이아웃 정상
- [ ] 대형 화면 (iPhone 15 Pro Max) 레이아웃 정상
- [ ] 가로 모드 처리 (잠금 또는 적응)

## 9. 서비스 오픈 정책

- [ ] 외부 앱 설치 유도 문구/배너 없음
- [ ] 외부 링크 이동 없음 (법률 페이지 제외)
- [ ] 앱 내 기능이 미니앱에서 완결적으로 동작
- [ ] AI 생성 콘텐츠 사용 시 고지/표시 의무 충족
