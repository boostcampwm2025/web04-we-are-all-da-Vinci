## 📖 Service Introduction

**우리 모두 다빈치**는 실시간으로 여러 명의 플레이어가 함께 즐길 수 있는 그림 기억 게임입니다.

### 게임 방식

1. 🎯 **제시된 그림 관찰**: 짧은 시간 동안 공개되는 그림을 집중해서 봅니다
2. 🎨 **기억을 바탕으로 그리기**: 사라진 그림을 최대한 똑같이 그립니다
3. 🏆 **유사도 채점 및 순위 확인**: 유사도 채점 알고리즘으로 원본과의 유사도를 분석하여 점수를 매깁니다
4. 👑 **최종 우승자 결정**: 가장 높은 점수를 받은 플레이어가 `오늘의 다빈치!`

실시간으로 다른 플레이어들의 그림과 점수를 확인하며 경쟁하는 재미를 느껴보세요!

## ✨ Key Features

### 🎮 실시간 멀티플레이

- **Socket.io 기반 실시간 통신**으로 끊김 없는 플레이 경험
- 방 생성 및 입장을 통해 최대 8명까지 함께 플레이 가능
- 실시간 랭킹 시스템으로 다른 플레이어와 경쟁

### 🤖 그림 채점

- 유사도 알고리즘을 활용한 정확한 그림 유사도 분석
- 공정하고 객관적인 점수 산정
- 실시간으로 채점 결과 확인

### 🎨 직관적인 드로잉 인터페이스

- 사용하기 쉬운 드로잉 툴 (브러시, 지우개, 색상 선택 등)
- 부드러운 그리기 경험 제공

### ⚙️ 커스텀 게임 설정

- 라운드 수, 제한 시간, 최대 플레이어 수 등 자유롭게 설정
- 내 스타일에 맞는 게임 방 생성 가능

### 📊 실시간 게임 현황

- 현재 진행 중인 라운드와 남은 시간 표시
- 실시간 순위표로 경쟁 현황 파악
- 게임 종료 후 최종 결과 및 순위 확인
