## 우리 모두 다빈치 🎨

짧은 시간 동안 공개된 그림을 기억하고, 기억을 바탕으로 **최대한 유사하게 그리는 사람이 승리**하는 실시간 멀티플레이 그림 게임입니다.  
비대면 상황에서도 팀원들과 즐겁게 소통하며, 웃음과 경쟁을 동시에 느낄 수 있어요!
